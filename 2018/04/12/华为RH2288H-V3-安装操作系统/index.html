华为RH2288H V3 安装操作系统

发表于 2018-04-12 | 更新于: 2019-11-09 | 阅读量 次

华为RH2288H V3服务器重新安装Windows Server 2008 R2操作系统，系统安装在一块独立的硬盘上，此方法不适合将系统安装在Raid阵列上。

安装系统前不要插任何第三方的卡！！！

安装系统驱动前不要插任何第三方卡！！！

安装操作系统

  1. 首先制作启动盘，本次使用工具为WinSetupFromUSB-1-8。

  2. 制作完启动盘后将U盘插入服务器USB口（网上言论说多数U盘会不识别，但我的正常识别：闪迪32GB USB3.0）。

  3. 开机，等待到达BIOS启动项界面

    说明：

    ​ 界面参数说明如下：

    按“F5”选择键盘类型，可选择键盘类型包括美式键盘和法式键盘。
    按“Delete”/“F4”进入BIOS Setup Utility界面。
    按“F11”/“F3”进入启动管理界面，在此界面可以选择启动设备进行启动。
    按“F12”，从网络启动快捷方式。

  4. 进入启动管理界面后，找到U盘设备启动。

  5. 接下来就跟普通装系统一样了。

安装驱动

  1. 到华为网站下载“FusionServer_iDriver-Windows-Driver-V329.zip”（版本会有变化）
  2. 解压进入路径“FusionServer_iDriver-Windows-Driver-V329.zip\Windows-Driver-V329\win2k8r2sp1”
  3. 解压“onboard_driver_win2k8r2sp1.iso”，进入文件夹“onboard_driver_win2k8r2sp1”
    1. 打开cmd，切换到驱动程序文件的路径
    2. 输入以下命令安装所有驱动程序：install.bat install “all”
    3. 安装完成后，您可以在工作路径下找到result.txt。在该文件中，您可以找到该操作的结果和信息。
  4. 安装后，网卡、USB3.0、Raid卡被识别（应该还有其他被识别的驱动）。在设备管理器中还有许多未被识别的设备。
  1. 到华为网站下载“FusionServer_Tools-ServiceCD2.0-V132.zip”（版本会有变化）
  2. 解压后进入文件夹，使用虚拟光驱工具加载“FusionServer Tools-ServiceCD2.0-V132.iso”。切记不可直接解压运行，直接解压运行会报错。
  3. 加载虚拟光驱后运行“InstallWinDriver.exe”，安装驱动，重启。
  4. 重启过后所有系统驱动即可安装完毕。
